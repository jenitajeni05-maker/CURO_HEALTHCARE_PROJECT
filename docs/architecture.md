# CURO System Architecture

## Components
1. Wearable IoT Smart Watch  
2. Hospital Alert Dashboard  
3. Family Notification System  
4. Cloud Database  
5. GPS Tracking System  

## Workflow
1. Device continuously monitors heart rate
2. System learns normal heart behaviour
3. Abnormal heartbeat detected
4. Alert sent to cloud server
5. Notification sent to nearby hospitals
6. Family members receive emergency alert
7. Ambulance dispatched to patient location

## Hardware (Proposed)
- Heart Rate Sensor (PPG)
- Microcontroller (ESP32 / Arduino)
- GPS Module
- GSM / Internet Module

## Software (Proposed)
- Mobile App for Patients
- Hospital Web Dashboard
- Cloud Database
- Real-time Notification System
